# IntentRadar

AI-powered social media intent monitoring platform that helps you discover and engage with potential customers across multiple social platforms.

## Features

- **AI-Powered Onboarding**: Two-mode configuration system (Chat & Form)
- **Unified Intent Inbox**: Monitor intents from Xiaohongshu, LinkedIn, X, Reddit, and more
- **Smart Intent Scoring**: AI evaluates each intent to help you prioritize
- **Analytics Dashboard**: Track metrics and trends across platforms
- **Multi-Platform Support**: Integration with major social networks

## Tech Stack

- Next.js 15 (App Router)
- React 19
- Tailwind CSS v4
- shadcn/ui
- Vercel AI SDK
- TypeScript

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000)
